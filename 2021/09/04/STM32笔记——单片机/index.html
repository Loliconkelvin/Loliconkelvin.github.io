新建一个工程（详细）

  1. 在总工程文件夹下创建子工程文件夹，文件名任意，keil工程文件名为project，子文件夹下创建 StartUp User Library 三个文件夹


  2. 搜索 103C8 ，弹出界面叉掉


  3. 编码格式改为 UTF-8

———————————-User———————————–

  1. 在 User 文件夹下创建mian.c，包含头文件，写一个main函数和一个死循环

    • （基于库函数）复制

      E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Project\STM32F10x_StdPeriph_Template

    中的 stm32f10x_conf.h stm32f10x_it.c stm32f10x_it.h 至 User

———————————StartUp———————————

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\CMSIS\CM3\DeviceSupport\ST\STM32F10x\startup\arm

中的 startup_stm32f10x_md.s 详见STM32容量与型号至 StartUp

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\CMSIS\CM3\DeviceSupport\ST\STM32F10x

中 stm32f10x.h system_stm32f10x.c system_stm32f10x.h 至 StartUp

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\CMSIS\CM3\CoreSupport

中的 core_cm3.c core_cm3.h 至 StartUp

———————-（基于库函数）Library————————

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\STM32F10x_StdPeriph_Driver\inc

中的所有文件至 Library

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\STM32F10x_StdPeriph_Driver\src

中的所有文件至 Library

0-1 用USB-TTL烧录软件(F103C8T6)

参考：
① 通过USB转TTL串口下载stm32程序
② 使用USB转TTL串口板和ST-LINK调试下载器给STM32单片机下载程序

———————————准备工作———————————

  1. USB转TTL供电跳线帽接到 VCC - 5V 让5V给转换器供电，3.3V给STM32


  2. STM32上面两个跳线帽 调整为 B0+ B1- （默认B0- B1-）

———————————接线烧录———————————

  1. 按照下面表格接线

    USB-TTL STM32
    3.3V V3
    TXD PA10
    RXD PA9
    GND G

  2. （先装一个USB-TTL驱动）使用软件烧录: 软件下载


    选择USB-TTL在PC的端口 -> NEXT -> download to device -> 选择编译的HEX文件 -> NEXT -> 去喝杯茶

———————————硬件复原———————————

  1. STM32上面两个跳线帽 调整为 B0- B1-


  2. 将USB转TTL跳线帽接回5V


  3. 复位单片机,测试程序

1-1 GPIO——LED闪烁

1-1-1 LED闪烁

  1. 使能IO口时钟

    • 找到RCC_APB2时钟控制函数（stm32f10x_rcc.h拉到最下面，找693行RCC_APB2PeriphClockCmd）并使用
    • 右键函数，Go to the definition of the…… 跳转到函数定义
    • 往上滚动，选择外设（如点亮PA0口则需要使能GPIOA，选择RCC_APB2Periph_GPIOA），放到第一个参数
    • 第二个参数也可以在定义上方找到，选择Enable，放到第二个参数

  2. 初始化GPIO

    • 找到GPIO_init函数（stm32f10x_gpio.h拉到最下面，找351行GPIO_Init）并使用
    • 右键函数跳转到定义
    • 第一个参数是GPIOx，x可以是A-G
    • 第二个参数是一个结构体，先把结构体数据类型复制，粘贴指代码（注意不要把指针的星号复制过去）
    • 定义一个名为GPIO_InitStructure的结构体，将其地址作为GPIO_init的第二个参数（取地址符&）
      • 用“.”引出结构体成员 GPIO_Mode GPIO_Pin GPIO_Speed
    • 右键成员跳转到定义，可以看到成员右边的注释：该成员可以是 xxxx_typedef 中的一个值，选中 xxxx_typedef 按 ctrl + f 搜索 -> find next
      • 若找到多个定义，选择 class为menber 的结果
    • 上滚找到枚举，复制其中一个作为成员的值

  • GPIO_Mode

    成员值 说明
    GPIO_Mode_AIN 模拟输入（Analog In）
    GPIO_Mode_IN_FLOATING 浮空输入
    GPIO_Mode_IPD 下拉输入（In Pull Down）
    GPIO_Mode_IPU 上拉输入（In Pull Up）
    GPIO_Mode_Out_OD 开漏输出（Out Open Drain）
    GPIO_Mode_Out_PP 推挽输出（Out Push Pull）
    GPIO_Mode_AF_OD 复用开漏（Atl Open Drian）
    GPIO_Mode_AF_PP 复用推挽（Atl Push Pull）

  • GPIO_Pin
    对应想要控制的引脚号，或全部引脚

  • GPIO_Speed
    对应速度，一般选择50MHz的

  1. GPIO输出函数

    • GPIO_SetBits(GPIO_TypeDef* GPIOx, uint16_t GPIO_Pin);
      可以将指定组的指定引脚设定为高电平
      第一个参数是GPIOx，x可以是A-G
      第二个参数是GPIO_Pin_x，n可以是0-15


    • GPIO_ResetBits(GPIO_TypeDef* GPIOx, uint16_t GPIO_Pin);
      可以将指定组的指定引脚设定为低电平


    • GPIO_WriteBit(GPIO_TypeDef* GPIOx, uint16_t GPIO_Pin, BitAction BitVal);
      可以将指定组的指定引脚设定为高/低电平（取决于第三个参数）
      第三个参数可以通过赚转到函数定义查看，可以是下面两个值：

      值 电平
      Bit_SET 高电平
      Bit_RESET 低电平
