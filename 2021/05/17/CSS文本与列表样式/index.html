CSS文本与列表样式

发表于 2021-05-17 更新于 2021-05-18 分类于 CSS

CSS文本与列表样式

字体组（font）

字体组合并写法

font:font-style font-variant font-weight font-size line-height font-family;

  • line-height 行高（详见不属于任何组-行高）

  • 注意：

    • 若只有一项要设置的值，不要用合并写法，使用font-写法
    • font属性合并写法必须要有 font-size 和 font-family 这两个值
    • 顺序必须严格遵守上述属性顺序
    • 若需要同时设置字体大小和行高，使用斜杠分隔这两项

      font: bold 12px/2.0em Verdana;

字体风格（style）

  • normal:默认
  • italic:应用字体的斜体属性
  • oblique:显示倾斜的字体

    部分字体没有斜体属性，应用italic会没有效果，可以使用oblique以显示倾斜的字体

设置小型大小写字母（variant）（不常用）

  • normal:默认
  • small-caps:显示小型大写字母

字体粗细（weight）

  • 该选项调节所选文字的粗细
值 效果
normal 普通
lighter 细
bold 粗
bolder 嗨粗

字体大小（size）

  • 该选项调节所选文字的字号

字体（family）

  • 该选项调节所选文字的字体
  • 可以设置多种字体，系统会逐个尝试，若不支持第一种字体则会尝试下一个
  • 注意：
    • 设置多字体时，每个字体名用逗号隔开
    • 字体名称包含空格、#、$等符号需要用双引号包含整个字体名称
    • 当需要设置英文字体时，英文字体需要位于中文字体之前

文本组（test）

垂直对齐方式（align）

  • left：左对齐
  • center：居中
  • right：右对齐

首行缩进（indent）

缩进2字符（em表示字符数 px表示像素）

装饰线（decoration）

underline:下划线
line-through:删除线（中划线）
overline:上划线
none:无（通常用于超链接去下划线）

文本阴影（shadow）

  • 用于指定文本阴影
  • 四个值分别对应水平阴影，垂直阴影，模糊距离，阴影颜色

不隶属于任何组

文本颜色（color）

  • 直接使用color即可设定字体颜色，他不隶属于font
  • color有四种设定值：
    • 单词设定
      color:red;
    • Hex RGB
      color:#00FF00;
    • RGB
      color:RGB(0,255,255);
    • RGBA（加入了透明度）
      color:RGBA(0,255,255,0.5)
      • 透明度的范围为0~1

行高（line-height）

  • 通常用于字体上下居中，字体默认固定在行高的一半处，那么只要行高与某格子的高度一样，那么字体就可以在那个格子里面上下居中

水平对齐（vertical-align）

  • text-top: 把该元素与父元素字体的顶端对齐
  • middle：把该元素放置在父元素的中部
  • text-bottom：把该元素的底部与父元素字体的底部对齐
  • 百分比：允许负值，负值会使该元素降低

列表组（list）

列表风格（style）

  • none：无（通常用于去除序号与圆点）
  • circle：空心圆
  • decimal：数字
  • 正方形：square
