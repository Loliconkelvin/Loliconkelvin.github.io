发表于 2021-04-27 更新于 2021-05-17 分类于 CSS

CSS的3+1种导入方式

行内样式（样式优先级最高）

  • 所谓行内样式，即直接在HTML标签中设置样式
  • 不建议使用，建议只在测试时使用
    <h1 style="color: red;"></h1>

内部样式表（优先级：就近原则）

  • 在HTML文件的head标签包含范围内写style标签，style标签内写CSS代码
    <head>
    <style>
        h1{
            color: red;
        }
    </style>
</head>

外部样式表（优先级：就近原则）

  • 在HTML文件外创建CSS文件，再于HTML文件内使用语句导入CSS文件

    链接式

  • 使用link标签
  • 与导入式的区别：
    • 载入网页时优先加载CSS文件至页面
      <link rel="stylesheet" herf="路径"/>

导入式（不建议使用）

  • 使用style标签与@import
  • 与链接式的区别：
    • 载入网页时先加载HTML文件再载入CSS文件
      <style type="text/css">
    @import url("路径");
</style>

样式优先级

  • 行内样式优先级最高
  • 内部样式与外部样式：
    • 看style标签与link标签哪个相比于正文（相比于body）近
