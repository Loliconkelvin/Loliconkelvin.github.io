使用下划线分隔超大数字

  • Java 7 以后，数字文字中数字之间的任意位置都可以出现任何数量的下划线字符，通过下划线你可以把数字按组进行分开。从而可以可以提高代码的可读性。
  • 注意不能在以下位置添加分割线：
    1. 数字的开头或结尾
    2. 浮点数中与小数点相邻的位置
    3. 后缀 F L D 的前一个位置

运算符——与或运算——短路运算

  • 与运算：算符两边都为真即为真

  • 或运算：算符两边都为假即为假

  • 因此当算符前面的值就不符合要求的时候

    与运算第一个就是假
    或运算第一个就为真

    运算符后面的指令不会执行（不会判断）

运算符——位运算——左移右移

  • 位运算效率极高
  • 遇到要循环*2或者循环/2的时候可以使用位运算
