HTML列表与表格

发表于 2021-04-21 更新于 2022-06-19 分类于 HTML

列表与表格（本章表格章节开始不再写快速输入有关内容）

列表

有序列表（orderlist）

  • 可使用 ol + tab/enter li + tab/enter 快速输入有序列表
  • 有序列表在生成时自带序号
<ol>
    <li>第一项</li>
    <li>第二项</li>
    <li>第三项</li>
    <li>第四项</li>
    <li>第五项</li>
</ol>

无序列表（unorderlist）

  • 可使用 ul + tab/enter li + tab/enter 快速输入无序列表
  • 无序列表在生成时自带原点
<ul>
    <li>项</li>
    <li>项</li>
    <li>项</li>
    <li>项</li>
    <li>项</li>
</ul>

自定义列表（definelist）

  • 可使用 dl + tab/enter dt + tab/enter dd + tab/enter 快速输入无序列表
  • 自定义列表包含三部分：
    1. dl：总的列表标签
    2. lt：title，表示列表标题（可以有多个列表标题，对应列表下面的内容）
    3. dd：列表内容
<dl>
    <dt>这是第一个列表标题</dt>
    <dd>内容</dd>
    <dd>内容</dd>
    <dd>内容</dd>
    <dd>内容</dd>
    <dd>内容</dd>
    <dt>这是第二个列表标题</dt>
    <dd>内容</dd>
    <dd>内容</dd>
    <dd>内容</dd>
    <dd>内容</dd>
    <dd>内容</dd>
</dl>

表格

  • 表格包含三个内容：
    1. 总表格标签（table）
    2. 行标签（tablerow）
    3. 列标签（td）
    • 列标签一定要在行标签内
<table>
    <tr>
        <td>1-1</td>
        <td>1-2</td>
        <td>1-3</td>
    </tr>
    <tr>
        <td>2-1</td>
        <td>2-2</td>
        <td>2-3</td>
    </tr>
    <tr>
        <td>3-1</td>
        <td>3-2</td>
        <td>3-3</td>
    </tr>
</table>

▽跨行与跨列都要在列标签中操作

跨行

  • 使用 rowspan="要跨的行数" 来实现跨行
  • 在跨行之后在剩余每行删除已跨的行数

跨列

  • 使用 colspan="要跨的列数" 来实现跨列
  • 在跨行之后在该行删除已跨的列数

border值

  • 把 border="1px" 应用到table标签，可以实现边框
