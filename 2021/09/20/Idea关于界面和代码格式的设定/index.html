发表于 2021-09-20 更新于 2022-07-15 分类于 Idea

IDEA 首选项设置

界面

工具条靠左显示

效果：

更改这两项：

代码格式

设置换行符为 Unix 编码格式

  • Settings -> Editor -> Code Style -> Line Separator 设置为 Unix and macOS(\n)

设置缩进

  • Settings -> Editor -> Code Style -> Java -> Tab and Indents
    1. 不勾选 Use tab character
    2. 下面的数字设置应该为 4 4 8 0

强制类型转换不带空格

  • Settings -> Editor -> Code Style -> Java -> Spaces -> other -> After type cast 设置为 不勾选

UTF-8

  • Settings -> Editor -> File Encodings 1. Globle Encoding 设置为 UTF-8 2. Project Encoding 设置为 UTF-8 3. Default encoding for properties file 4. 勾选 第三条旁边的 Transparent native-to-ascii conversion 5. Create UTF-8 files 设置为 with no BOM

禁用双 shift 全局搜索

如何关掉使用 IDEA 连续按两次 Shift 键弹出的全局搜索

启用序列化 uid 警告

  • Settings 搜索 Serializable class，勾选 Serializable class without “serializationUID”
