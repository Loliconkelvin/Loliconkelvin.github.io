常用DOS命令

发表于 2021-04-16 更新于 2022-06-19 分类于 DOS

常用DOS指令

打开CMD

  1. 开始菜单 → 命令提示符
  2. win + R 运行 → CMD
  3. 在任意文件夹下 shift + 右键 → 在此处打开命令行窗口
  4. 资源管理器 → 地址栏 → 输入 CMD + 空格 + 运行路径

管理员身份运行

  1. 开始菜单 → 命令提示符处右键 → 以管理员身份打开

粘贴指令

  • 在复制指令之后，右键终端即可粘贴指令

常用DOS命令

  • 带*的功能有注释
功能 指令格式 实例
切换盘符 盘符 + 冒号 E:
遍历文件（夹） dir dir
切换目录* cd + 参数 + 目录 cd /d F:\tool
清楚终端屏幕 cls cls
查看电脑IP ipconfig ipconfig
打开Win自带常用工具* 工具名 calc
测试网址是否正常 ping + 空格 + 网址 ping www.baidu.com
创建文件夹 md + 空格 + 文件夹名 md test
在指定下一级目录新建文件 cd + > + 文件名（带后缀） cd>test.txt
删除文件 del + 空格 + 文件名（带后缀） del test.txt
删除文件夹（不经过回收站） rd + 空格 + 文件夹名 rd test
退出终端 exit exit

  • 切换目录

    1. 跨盘切换目录需要在目录前加入 /d
    2. 返回到上一级，可以使用 ..
    3. 进入到下一级目录可以直接在cd后加入文件名
