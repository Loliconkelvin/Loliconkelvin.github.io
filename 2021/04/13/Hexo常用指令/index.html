Hexo常用指令

新建文章

$ hexo new layout "blogtitle"
  • new 可以简写为n
  • layout 文章类型，可以不指定，默认类型为post
  • blogtitle 文章标题

清理缓存文件

$ hexo clean

生成静态文件

$ hexo generate
  • generate 可以简写为g
  • 生成后部署可以使用 hexo g -d

启动本地服务器

$ hexo server
  • server 可以简写为s
  • -p 指定服务器端口，不指定默认为4000
  • -i 指定服务器IP地址，不指定默认为0.0.0.0，即localhost
  • -s 静态模式，禁用文件监视

部署（上传）

$ hexo deploy
  • deploy 可以简写为d
