发表于 2021-04-13 更新于 2022-06-19 分类于 Hexo

Hexo、npm、NexT基本设置

npm基本设置

卸载所有全局模块

  • 查看所有全局安装的模块

    npm ls -g

  • 卸载全局模块

    npm uninstall 包名 -g

更改全局模块安装和缓存目录

  • 显示npm当前设置
# 显示部分设置
npm config ls

# 显示全部设置
npm config ls -l

全局模块默认安装于 C:\Users\用户名\AppData\Roaming\npm

  • 更改全局模块安装目录
npm config set prefix "E:\Blog Tools\Nodejs\node_modules\npm_global_modules"

执行完之后重启gitbash，输入 hexo -version 验证

  • 更改全局缓存目录
npm config set cache "E:\Blog Tools\Nodejs\node_modules\npm_cache"

更换软件源（可选）

npm config set registry "https://registry.npm.taobao.org"

全局安装Hexo

  • 安装Hexo
npm install hexo-cli -g
  • 建站

若之前就已经创建好，则下面的不用做

# 初始化博客文件夹
hexo init <folder>

# 移动到博客文件夹
cd <folder>

# 安装Hexo插件
npm install

# 安装其他插件
npm install hexo-server --save
npm install hexo-admin --save
npm install hexo-generator-archive --save
npm install hexo-generator-feed --save
npm install hexo-generator-search --save
npm install hexo-generator-tag --save
npm install hexo-deployer-git --save
npm install hexo-generator-sitemap --save

升级插件

  • 安装 npm-check

    npm install -g npm-check

  • 检查更新

    # 检查当前项目下（局部）插件更新
npm-check -u

# 检查全局插件更新
npm-check -u -g

使用上下左右控制绿色指针
使用空格选择
使用 Enter 更新所选插件
ctrl + c 退出

清除npm缓存

npm cache clean -f

NexT基本设置

参考官方配置文档：http://theme-next.iissnan.com/getting-started.html

开启本地服务器

hexo server -debug

设置主题

  1. 前往 Blog\_config.yml 搜索 theme

  2. 在最后加上 theme: next

  3. 将其他主题设置注释掉，一般就在附近一两行

  4. 前往网页刷新验证

设置语言

  1. 前往 Blog\_config.yml 搜索 language

  2. 设置为 language: zh-CN （注意是 zh-CN 不是 zh-Hans）

设置菜单

  1. 前往 Blog\node_modules\hexo-theme-next 搜索 menu

  2. 注释代表不激活，不注释即为激活，在这里可以设置：

    • home：主页按钮
    • about：关于页面按钮
    • tags：标签页面按钮
    • categories：分类页面按钮
    • archives：归档页面按钮
    • schedule：日程表页面按钮
    • sitemap：站点地图页面按钮
    • commonweal：公益404页面按钮

    目前的设置激活的有：home tags categories

设置头像

  1. 前往 Blog\node_modules\hexo-theme-next 搜索 avatar

  2. 设置URL

    • 你可以设置完整的网络URL，如：avatar: http://example.com/avatar.png

    • 也可以设置本地相对路径
      注意1：相对路径以Blog\source 目录为根目录
      注意2：请注意斜杠方向

    目前的设置为：url: /uploads/avatar.png
    即 Blog\source 文件夹下有一目录 uploads
    uploads 文件夹下存放头像 avatar.png

设置代码高亮主题

从这里开始参考官方文档-主题配置：http://theme-next.iissnan.com/theme-settings.html

  1. 前往 https://theme-next.js.org/highlight/ 选择一个主题

  2. 更换引擎：前往 Blog\_config.yml 搜索 highlight 在这里可以更换 Highlight.js 引擎和 PrismJs 引擎，根据上一步所选择的主题的引擎，将引擎名下一行的 enable 分别设置为 true 和 false

  3. 更换高亮主题：前往 Blog\node_modules\hexo-theme-next 搜索 codeblock

  4. 将对应主题下的 light 和 dark 后面都加上主题名

  • 如需开启复制按钮，在主题配置文件的高亮主题设置下面有一个 copy_button 设置为 enable: true

目前的设置为：
引擎：PrismJs
主题：prism-okaidia
复制按钮：启用

侧边栏社交链接

  1. 前往 Blog\node_modules\hexo-theme-next 搜索 social

  2. 反注释需要用的社交链接，并且修改后面的链接地址

目前的设置开启的有 GitHub Email

载入页面时的动画效果

  1. 前往 Blog\node_modules\hexo-theme-next 搜索 motion

  2. 若需要修改动画效果，前往 https://theme-next.js.org/animate/ 或 https://animate.style/ 选择喜欢的动画，将名字复制到对应的组件动画后，可以设置的组件有：

    1. post_block：主页中每个文章的占用的方块 或 文章与文章之间的分割线
    2. post_header：文章标题
    3. post_body：文章正文 或 <!-- more --> 前面的内容 和 阅读全文按钮
    4. coll_header：页眉
    5. sidebar：主页左侧的工具条 和 每篇文章的目录

目前的设置开启的有 post_header post_body sidebar
动画为默认动画

背景动画

  1. 前往 Blog\node_modules\hexo-theme-next 搜索 Canvas ribbon

  2. 将下面的 enable 设置为 true

目前的设置为开启

回到顶部按钮的一点小个性化

  1. 前往 Blog\node_modules\hexo-theme-next 搜索 back2top

  2. 回到顶部功能默认开启，在下面的 scrollpercent 后面可以启用当前阅读位置的比例显示

目前的设置为比例显示开启

Hexo常用指令

新建文章

hexo new layout "blogtitle"
  • new 可以简写为n
  • layout 文章类型，可以不指定，默认类型为post
  • blogtitle 文章标题

清理缓存文件

hexo clean

生成静态文件

hexo generate
  • generate 可以简写为g
  • 生成后部署可以使用 hexo g -d

启动本地服务器

hexo server
  • server 可以简写为s
  • -p 指定服务器端口，不指定默认为4000
  • -i 指定服务器IP地址，不指定默认为0.0.0.0，即localhost
  • -s 静态模式，禁用文件监视

部署（上传）

hexo deploy
  • deploy 可以简写为d
