vscode用户自定义代码块

发表于 2021-04-24 更新于 2022-03-26 分类于 vscode

在Markdown重复输入某些固定格式的html标签是一件很烦的事，这时候可以使用到vscode用户自定义代码块功能

一、自定义代码块的创建

设置->用户代码片段
选择一种语言

按照以下格式写一个自定义代码块

"sheet": {
	"prefix": "sheet",
	"body": [
		"| $1 |",
		"| :--: |",
		"| $2 |"
	],
	"description": "创建表格"
}

注意：

1. “sheet”: {} 外面的sheet表示代码块名

2. prefix后的值表示输入代码块所需要的触发前缀

3. body后的值为代码块

4. description后的值为代码片段说明

二、代码块的编辑

代码块一行一句，每句用逗号结尾

转义字符

  • 使用转义字符\t实现tab缩进
  • 使用转义字符\n实现换行
  • 遇到需要输入双引号时需要转义\"
  • 空格可以直接打

预设光标位置

  • 使用$1 $2 $3 $0等可以预设光标的位置
  • $0是光标的最后位置
  • 在快速输入后，使用tab可以在光标预设位置之间跳转

三、启用自定义代码块提示

进入vscode设置（代码模式）

加入以下代码：

若不是加在最后，注意要在}后加一个逗号

"[markdown]":{
 "editor.quickSuggestions": true
}
