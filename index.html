STM32笔记

发表于 2021-09-04 分类于 STM32

点灯大师

0-0 范例工程

新建一个工程（详细）

  1. 在总工程文件夹下创建子工程文件夹，文件名任意，keil工程文件名为project，子文件夹下创建 StartUp User Library 三个文件夹


  2. 搜索 103C8 ，弹出界面叉掉


  3. 编码格式改为 UTF-8

———————————-User———————————–

  1. 在 User 文件夹下创建mian.c，包含头文件，写一个main函数和一个死循环

    • （基于库函数）复制

      E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Project\STM32F10x_StdPeriph_Template

    中的 stm32f10x_conf.h stm32f10x_it.c stm32f10x_it.h 至 User

———————————StartUp———————————

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\CMSIS\CM3\DeviceSupport\ST\STM32F10x\startup\arm

中的 startup_stm32f10x_md.s 详见STM32容量与型号至 StartUp

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\CMSIS\CM3\DeviceSupport\ST\STM32F10x

中 stm32f10x.h system_stm32f10x.c system_stm32f10x.h 至 StartUp

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\CMSIS\CM3\CoreSupport

中的 core_cm3.c core_cm3.h 至 StartUp

———————-（基于库函数）Library————————

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\STM32F10x_StdPeriph_Driver\inc

中的所有文件至 Library

  1. 复制

    E:\Keil5\STM32F10x_StdPeriph_Lib_V3.5.0\Libraries\STM32F10x_StdPeriph_Driver\src

中的所有文件至 Library

———————————–收尾————————————-

  1. 将 StartUp User Library 三个文件夹和里面的文件包含进工程


  2. 魔术棒 -> C/C++ -> Include Path加入 StartUp User Library 三个路径


  3. 魔术棒 -> C/C++ -> Define

    加入 USE_STDPERIPH_DRIVER（该字符串在头文件9301行）


  4. （有使用ST-link连接单片机时）

    魔术棒 -> Debug -> Use 设置为ST-link


  1. （有使用ST-link连接单片机时）

    Use旁边的Settings -> Flash Download

    勾选Reset and Run


  2. 工具栏 -> 箱子

    把User移到最下面


  3. 尝试编译
