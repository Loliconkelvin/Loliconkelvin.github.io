CSS图片与渐变

背景组（background）

背景组合并写法

background : background-color background-image background-position/background-size background-repeat background-attachment;

背景颜色（color）

  • 不解释

背景图片（image）

div{
    background-image: url("");
}

背景图起始位置（position）

  • 百分数表示：x% y%
  • 像素表示：xpx ypx

平铺方式（repeat）

  • no-repeat: 不平铺
  • repeat-x: 横向平铺
  • repeat-y: 纵向平铺

固定或滚动（attechment）

  • fixed：背景图片不随着页面滚动而滚动

    打开网页看到是怎么样的，滚动后看到就是怎么样的

  • scroll：背景图片随页面滚动而滚动，但不随着元素滚动而滚动（默认）

    图片不作为整一个页面的背景图，而作为页面内一个带有滚动元素的元素的背景图
    滚动网页可以看到图片下面的部分

  • local：背景图片随着元素内容的滚动而滚动，也随着网页滚动而滚动
