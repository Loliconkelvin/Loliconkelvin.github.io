CSS定位

定位（position）

相对定位（relative）

  • 元素相对于原来的位置偏移
  • 元素还在一般文本流里面，父级外框会给元素留位置

偏移

  • 只使用position不能使元素偏移，需要搭配top left right bottom
  • 这四个元素指距离上、做、下、右距离，因此要向这个方向移动，需要把值设为负数

绝对定位（absolute）

  • 没有父级元素定位的情况下，元素相对于浏览器边框偏移，并且滚动后该元素会保持其绝对位置不变
  • 若父级元素有position，相对于父级元素边框位移（在父级元素范围内活动）（常用）
  • 元素不在一般文本流里面，父级元素不再给它留位置

固定定位（fixed）（回到顶部）

  • 相对于浏览器边框偏移，并且无论怎么滚动，保持其相对于浏览器边框的位置不变

图层（z-index）

  • 在使用z-index之前需要应用position
  • 默认值为0，即置于最底层，最高无限
