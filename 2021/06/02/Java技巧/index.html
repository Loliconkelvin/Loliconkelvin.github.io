使用下划线分隔超大数字

  • Java 7 以后，数字文字中数字之间的任意位置都可以出现任何数量的下划线字符，通过下划线你可以把数字按组进行分开。从而可以可以提高代码的可读性。
  • 注意不能在以下位置添加分割线：
    1. 数字的开头或结尾
    2. 浮点数中与小数点相邻的位置
    3. 后缀 F L D 的前一个位置

运算符——与或运算——短路运算

  • 与运算：算符两边都为真即为真

  • 或运算：算符两边都为假即为假

  • 因此当算符前面的值就不符合要求的时候

    与运算第一个就是假
    或运算第一个就为真

    运算符后面的指令不会执行（不会判断）

int num1 = 20;
int num2 = 10;
int var = 0;
System.out.println((num1 < num2) && (var++ < 3));
System.out.println(var);
// 因为num1 < num2为假，因此var不会自增

System.out.println((num1 > num2) || (var++ < 3));
System.out.println(var);
// 因为num1 > num3为真，因此var不会自增

运算符——位运算——左移右移

  • 位运算效率极高
  • 遇到要循环*2或者循环/2的时候可以使用位运算

字符串连接符

  • “ + ”可以用于字符串连接
int a = 10;
int b = 20;

System.out.println(a + b + "");
// 输出30
System.out.println("" + a + b);
// 输出1020

// 这里证明了当遇到String的时候，后面的值会变成String，此时+为字符串连接符，但在遇到String之前，会对值正常求和，此时为运算符

快捷for循环

  • 使用 循环次数.fori 创建自增循环 循环次数.forr创建自减循环

  • 可用于数组遍历

// 100.fori + 回车
for (int i = 0; i < 100; i++) {
        
}

// 100.forr + 回车
for (int i = 100; i > 0; i--) {
        
}

// numbers（数组名）.fori
for (int i = 0; i < numbers.length; i++) {

}

  • 使用 数组名.for 创建增强型for循环
// number（数组名）.for
for (int number : numbers) {
    
}

快速生成有参/无参构造器

  • 使用 Alt + Insert -> Constructor
    1. 选择 Select None 生成无参构造
    2. 选择属性生成以该属性为参数的有参构造
    3. 选择多个属性生成一个 有多个参数的无参构造

快速生成Getter/Setter（非lombok）

  • 使用 Alt + Insert -> Getter 为选定的属性分别生成Getter，请注意在return返回值处修改为 this.变量名
  • 使用 Alt + Insert -> Setter 为选定的属性分别生成Setter，无需修改
  • 使用 Alt + Insert -> Getter and Setter 为选定的属性分别生成Getter和Setter，注意在Getter处修改 this.属性名

切换编辑的文件

  • 当打开了多个文件时，使用 Alt + ← 或 Alt + → 切换编辑的文件
