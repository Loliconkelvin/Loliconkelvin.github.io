vscode常用快捷键

打开文件

ctrl + O

打开文件夹

ctrl + K ctrl + O

向下复制一行

alt + shift + ↓

上下移动一行

alt + ↑ alt + ↓

列编辑

ctrl + alt + ↑ ctrl + alt + ↓

回退光标操作

ctrl + U

查找

ctrl + F

整体缩进与取消缩进

ctrl + ] 缩进
ctrl + [ 取消缩进
