冒泡排序

  • 由于嵌套循环，冒泡怕羞的时间复杂度是O(n^2)

把大数往后推

public static void sort(int[] arrays) {
    int temp;

    if (arrays.length == 1 | arrays.length == 0) {
        return;
    }

    // 大循环,起点为最后一个数，该循环执行数组长度-1次
    for (int i = 0; i < arrays.length - 1; i++) {
        // 小循环 起点为第一个数和第二个数，第一个数作为基准,将大的数往后推
        for (int j = 0; j < arrays.length - 1 - i; j++) {
            // 对比&交换
            if (arrays[j] > arrays[j + 1]) {
                temp = arrays[j + 1];
                arrays[j + 1] = arrays[j];
                arrays[j] = temp;
            }
        }
    }
}

把小数往前推

public static void sort(int[] arrays) {
    int temp;

    if (arrays.length == 1 | arrays.length == 0) {
        return;
    }
    
    // 大循环,起点为最后一个数，该循环执行数组长度-1次
    for (int i = 0; i < arrays.length - 1; i++) {
        // 小循环 起点为最后一个和倒数第二个数，最后一个数作为基准,将小的数往前推
        for (int j = arrays.length - 1; j > i; j--) {
            // 对比&交换
            if (arrays[j] < arrays[j - 1]) {
                temp = arrays[j - 1];
                arrays[j - 1] = arrays[j];
                arrays[j] = temp;
            }
        }
    }
}

优化？

  • 减少无意义的排序
  • 可以定义一个变量作为标志，若有一次比较中没有进行任何交换，则说明数组已经排好序，可以跳出整个方法直接返回：
    public static void sort(int[] arrays) {
    int temp;
    boolean swap;

    if (arrays.length == 1 | arrays.length == 0) {
        return;
    }

    // 大循环,起点为最后一个数，该循环执行数组长度-1次
    for (int i = 0; i < arrays.length - 1; i++) {

        // 初始化标志变量：未进行交换
        swap = false;

        // 小循环 起点为最后一个和倒数第二个数，最后一个数作为基准,将小的数往前推
        for (int j = 0; j < arrays.length - 1 - i; j++) {
            // 对比&交换
            if (arrays[j] > arrays[j + 1]) {
                temp = arrays[j + 1];
                arrays[j + 1] = arrays[j];
                arrays[j] = temp;

                // 进行过交换，将标志变量定为true
                swap = true;
            }
        }

        // 若一轮对比下来没有进行过交换，则直接跳出循环并返回
        if (swap == false) {
            break;
        }
    }
}

稀疏数组

  • 对于二维数组， 数组名.length 返回的是行数， 数组名[下标].length 返回的是该行的列数（或者说该行的长度）

  • 当一个数组大部分元素为0，或者为同一数值时，可以使用稀疏数组来保存该数组，花时间省空间

  • 稀疏数组用于记录原始数组有几行几列、几个不同值

package com.company.demo;

public class Main {
    public static void main(String[] args) {
        // 创建一个二维数组
        int[][] numbers = new int[11][11];

        // 给个别元素赋值
        numbers[2][3] = 1;
        numbers[3][4] = 2;

        // 打印原始数组
        for (int[] numberRow : numbers) {
            for (int i : numberRow) {
                System.out.print(i+ " ");
            }
            System.out.println();
        }

        // 将原始数组转换为稀疏数组

        // 提取非零元素数
        int member = 0;
        for (int[] i : numbers) {
            for (int j : i) {
                if (j != 0) {
                    member++;
                }
            }
        }

        // 创建稀疏数组
        int[][] compressArray = new int[member + 1][3];

        // 表头：原数组行数 原数组列数 非零元素数
        compressArray[0][0] = numbers.length;
        compressArray[0][1] = numbers[0].length;
        compressArray[0][2] = member;

        // 写非零元素的坐标和值
        int memberNumber = 0;
        for (int i = 0; i < numbers.length; i++) {
            for (int j = 0; j < numbers[i].length; j++) {
                if (numbers[i][j] != 0){
                    memberNumber++;
                    compressArray[memberNumber][0] = i;
                    compressArray[memberNumber][1] = j;
                    compressArray[memberNumber][2] = numbers[i][j];
                }
            }
        }

        // 输出稀疏数组
        for (int[] i : compressArray) {
            for (int j : i) {
                System.out.print(j + " ");
            }
            System.out.println();
        }

        // 还原稀疏数组

        // 新建原始数组
        int[][] array = new int[compressArray[0][0]][compressArray[0][1]];

        // 根据稀疏数组给原始数组赋值
        for (int i = 1; i <= compressArray[0][2]; i++) {
            array[compressArray[i][0]][compressArray[i][1]] = compressArray[i][2];
        }

        // 输出原始数组
        for (int[] i : array) {
            for (int j : i) {
                System.out.print(j);
            }
            System.out.println();
        }
    }

}
