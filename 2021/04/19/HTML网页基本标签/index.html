HTML网页基本标签

发表于 2021-04-19 更新于 2022-07-18 分类于 HTML

HTML 网页基本标签

元素标签

标题标签

  • 可用 hx + tab/enter 快速输入
<h1>一级标签</h1>
<h2>二级标签</h2>
<h3>三级标签</h3>
<h4>四级标签</h4>
<h5>五级标签</h5>
<h6>六级标签</h6>

段落标签

  • 可以使用 p + tab/enter 快速输入
  • 段落标签有行距
<p></p>

span 标签（独立行内元素）

  • span 标签用于将某些元素从行元素中独立出来以使其与行元素中的其他元素应用不同的风格
  • 若不使用 style 属性，虽然 span 标签包围的元素确实从行元素中独立了出来，但是在显示时与不加 span 标签没有任何区别
  • span 标签因为其本身不起任何效果而经常用来定义锚点（因为使用 a 标签字体会变蓝）
<p>我是一个<span style="color:blue">男生</span></p>
<span id="urlInPage">我是锚点</span>

div 标签（整合多个元素）

  • div 标签会自动换行
  • div 标签用于整合多个元素
  • 对 div 标签应用 id 属性或 class 属性（或 style 属性直接修改样式），可以对 div 标签包含的内容批量修改样式

换行标签

  • 可以使用 br + tab/enter 快速输入（注意快速输入后不带 / 结束）
  • 换行标签没有行距
<br/>

水平线标签

  • 可以使用 hr + tab/enter 快速输入（注意快速输入后不带 / 结束）
<hr/>

字体样式标签

  • 可以使用 strong + tab/enter em + tab/enter 快速输入
    记粗体和斜体即可，其他可以用 CSS 写
<strong></strong>
<em></em>
或
<b></b>
<i></i>

补：字体样式标签——上标/下标

23
O2

上标
<sup>3</sup>

下标
<sub>2</sub>

特殊符号

符号 code 效果
空格 &nbsp; &nbsp;
大于号 &gt; >
小于号 &lt; <
注册商标 &reg; &reg;
版权符号 &copy; &copy;

图像标签

  • 可用 img + tab/enter 快速输入
  • 常用图像标签

    jpg gif png bmp ……

<img src="" alt="" title="" width="" height=""/>

src 属性（必填）

图片路径

  • 相对路径
    相对于本 HTML 文件路径（推荐使用）
<img src="../resources/image/1.jpg" alt="头像">
<!--
    .. 回到HTML文件的上级目录，即工程目录
    resources 资源文件夹
    image 图片文件夹
 -->
  • 绝对路径
    不推荐使用

alt 属性（必填）

  • 图片的替代名字，当图片无法加载时显示该属性的值

title 属性

  • 鼠标悬停上去时在鼠标附近显示的文字

width 和 height

  • 图片的宽和高

链接标签

超链接（页面间跳转）

  • 可使用 a + tab/enter 快速输入
  • 使用 <a>显示的内容</a> 来表示超链接
<a herf="http://www.baidu.com" target="">点击我跳转到百度</a>

a 标签内包含的内容

  • 要显示的文字
  • 要显示的图片（没错！只需要在 a 标签内包含一个图片标签 <img src="" alt=""/> 就可以实现点击图片跳转页面的效果

herf 属性（必填）

  • 表示要跳转到哪个页面
  • 可以使用完整的网址
  • 可以使用路径指向 HTML 文件
  • 可以使用锚链接

target 属性

  • 表示新页面在哪里显示
target 属性的值 效果
_blank 在新标签页打开新页面
_self 在本标签页打开新页面（无 target 时默认）
iframe 内联框架名 在内联框架内打开新页面

锚链接（页面内跳转）

  • 定义一个锚链接需要两步
    1. 定义一个标记
    2. 跳转

定义锚点

  • 使用 a 标签中的 name 属性设定标签
<a name="top">我是顶部</a>

跳转

  • 把 # + 锚点名 应用到 herf 属性即可跳转
<a herf="#top">点我回到顶部</a>

超链接+锚链接（跳转到某页面的指定位置）

  • 把 URL + # + 锚点名 应用到 herf 属性即可
<a herf="xxxxx.html#top"></a>

功能性链接

邮件链接

  • 把 mailto: + 邮箱地址 应用到 herf 属性即可
<a herf="mailto:1551107036@qq.com">点击联系我</a>

QQ 链接

  • QQ 推广 → 推广工具 → 复制代码

CSS

style 标签

<style></style>
  • style 标签内主要存放 css 语句

link 标签

<link rel="stylesheet" herf=""/>

herf 属性

  • 其值为 css 文件的路径
