HTML网页基本标签

发表于 2021-04-19 更新于 2021-05-17 分类于 HTML

HTML网页基本标签

元素标签

标题标签

  • 可用 hx + tab/enter 快速输入

段落标签

  • 可以使用 p + tab/enter 快速输入
  • 段落标签有行距

span标签（独立行内元素）

  • span标签用于将某些元素从行元素中独立出来以使其与行元素中的其他元素应用不同的风格
  • 若不使用style属性，虽然span标签包围的元素确实从行元素中独立了出来，但是在显示时与不加span标签没有任何区别
  • span标签因为其本身不起任何效果而经常用来定义锚点（因为使用a标签字体会变蓝）

div标签（整合多个元素）

  • div标签会自动换行
  • div标签用于整合多个元素
  • 对div标签应用id属性或class属性（或style属性直接修改样式），可以对div标签包含的内容批量修改样式

换行标签

  • 可以使用 br + tab/enter 快速输入（注意快速输入后不带 / 结束）
  • 换行标签没有行距

水平线标签

  • 可以使用 hr + tab/enter 快速输入（注意快速输入后不带 / 结束）

字体样式标签

  • 可以使用 strong + tab/enter em + tab/enter 快速输入
    记粗体和斜体即可，其他可以用CSS写

特殊符号

符号 code
空格 &nbsp;
大于号 &gt;
小于号 &lt;
版权符号 &copy;

图像标签

  • 可用 img + tab/enter 快速输入
  • 常用图像标签

    jpg gif png bmp ……

    src属性（必填）

    图片路径
  • 相对路径
    相对于本HTML文件路径（推荐使用）
  • 绝对路径
    不推荐使用

alt属性（必填）

  • 图片的替代名字，当图片无法加载时显示该属性的值

title属性

  • 鼠标悬停上去时在鼠标附近显示的文字

width和height

  • 图片的宽和高

链接标签

超链接（页面间跳转）

  • 可使用 a + tab/enter 快速输入
  • 使用 <a>显示的内容</a> 来表示超链接

a标签内包含的内容

  • 要显示的文字
  • 要显示的图片（没错！只需要在a标签内包含一个图片标签 <img src="" alt=""/> 就可以实现点击图片跳转页面的效果

herf属性（必填）

  • 表示要跳转到哪个页面
  • 可以使用完整的网址
  • 可以使用路径指向HTML文件
  • 可以使用锚链接

target属性

  • 表示新页面在哪里显示
target属性的值 效果
_blank 在新标签页打开新页面
_self 在本标签页打开新页面（无target时默认）
iframe内联框架名 在内联框架内打开新页面

锚链接（页面内跳转）

  • 定义一个锚链接需要两步
    1. 定义一个标记
    2. 跳转

定义锚点

  • 使用a标签中的name属性设定标签

跳转

  • 把 # + 锚点名 应用到herf属性即可跳转

超链接+锚链接（跳转到某页面的指定位置）

  • 把 URL + # + 锚点名 应用到herf属性即可

功能性链接

邮件链接

  • 把 mailto: + 邮箱地址 应用到herf属性即可

    QQ链接

  • QQ推广→推广工具→复制代码

CSS

style标签

  • style标签内主要存放css语句

link标签

herf属性

  • 其值为css文件的路径
