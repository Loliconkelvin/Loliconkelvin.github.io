iframe内联框架

发表于 2021-04-23 更新于 2022-07-18 分类于 HTML

iframe 内联框架

frameset

使用 frameset 代替 body 则表示该页面是由多个页面合并组成

一个完整的 framset 由两个标签组成：

  1. frameset 表示框架，用于承载在它里面的页面，也因此它可以设置这个框架应该怎样分成两块
  2. frame 主要用于指定显示哪个页面
<frameset>
  <frame src="" />
</frameset>

rows 上下分隔

要做到这样的效果，需要在 frameset 标签，加上 rows="n1, n2, ..." 属性

比如上面的效果的代码就是：

<!-- “*” 表示剩余的全部 -->
<frameset rows="40%,*">
  <frame src="top.html" />
  <frame src="bottom.html" />
</frameset>

cols 左右分隔

要做到这样的效果，需要在 frameset 标签，加上 cols="n1, n2, ..." 属性

比如上面的效果的代码就是：

<!-- “*” 表示剩余的全部 -->
<frameset cols="40%,*">
  <frame src="top.html" />
  <frame src="bottom.html" />
</frameset>

嵌套使用

可以使用 frameset 代替上面的 frame 将该框架应用于父框架的某个部分中，将该部分进行再分

<frameset cols="40%, *">
  <frame src="top.html" />
  <frameset rows="50%, *">
    <frame src="bottom.html" />
    <frame src="test.html" />
  </frameset>
</frameset>

iframe 内联框架

  • 又到了每个语言的套娃时间，如何在网页里面再显示 N 个网页呢
<iframe src="" name=""></iframe>

src 值

  • 你都要显示网页了，那这个肯定是网页 URL 啦
  • src 值不是必填，因为内联框架可以用于超链接

name

  • 给内联框架一个名字，可以让其他跳转指令的跳转效果在该内联框架实现

width 值

  • 定制内联框架的宽度

height 值

  • 定制内联框架的高度
  • 注意：
    1. 只在 iframe 设置 height 值将不能达到预期效果
    • 解决：
      将其父空间高度html与body的高度设置为 100%
      在 html 标签与 body 标签中增加以下属性：
      style="height:100%;"

frameborder 值（0/1）

  • 规定是否使用边框，0 为不使用，1 为使用
