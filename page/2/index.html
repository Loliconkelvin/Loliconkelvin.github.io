注释

行注释

  1. 说明：
    行注释对一行进行注释，适合注释字数少的时候使用
  2. 快速输入：
    无
  3. 代码规范：
    • 方法内部单行注释，在被注释语句上方另起一行，使用//注释。方法内部 多行注释使用/* */注释，注意与代码对齐。

段落注释

  1. 说明：
    段落注释允许写多行注释，适合注释字数多的时候使
  2. 快速输入：
    /* + enter
  3. 代码规范：
    无

方法注释（Javadoc）

  1. 说明：
    详见：https://blog.csdn.net/vbirdbest/article/details/80296136
  2. 快速输入：
    /** + enter
  3. 代码规范：
    • 类、类属性、类方法的注释必须要使用Javadoc规范，禁止使用行注释
    • 所有的抽象方法（包括接口中的方法）必须要用 Javadoc 注释,除了返回值、参数、异常说明外，还必须指出该方法做什么事情，实现什么功能。

标识符和关键字

标识符（含变量、方法、类命名风格）

  1. 说明：
    Java的所有组成部分都需要名字。类名，变量名和方法名都被称为标识符
  2. 代码规范：
    • 代码中标识符的命名均不能以下划线或美元符号开始，也不能以下划线和美元符号结束。
    • 禁止拼音英文混合命名和纯中文命名
    • 类名使用单词首字母大写形式。（如：XmlService）
    • 方法名、参数名、成员变量、局部变量使用驼峰命名。
    • 常量命名字母全部大写，单词间用下划线隔开。

      关键字

  3. 说明：
    Java关键字是电脑语言里事先定义的，有特别意义的标识符，有时又叫保留字，还有特别意义的变量。 Java的关键字对Java的编译器有特殊的意义，他们用来表示一种数据类型，或者表示程序的结构等，关键字不能用作变量名、方法名、类名、包名和参数。

数据类型

基本数据类型（八个）

整数类型 占用位数 范围
byte 8 -128 ~ 127
short 16 -32768 ~ 32767
int 32 2^32
long 64 2^64
  • 代码规范：
    • 使用long类型的时候必须要在数字后面加个大写L
浮点类型 占用位数
float 32
double 64
  • 代码规范：
    • 使用float类型的时候必须要在数字后面加个F
    • 使用double类型的时候必须要在数字后面加个D
  • 注意：
    • 尽量避免使用浮点类型进行比较
字符类型 占用位数
char 16
布尔类型 占用位数
boolean 8
