Idea/DataGrip激活

发表于 2022-06-09 更新于 2022-07-16 分类于 Idea

Idea/DataGrip 激活

  1. 点进这个网址中的任意一个可用网址：

    3.jetbra.in （← 目前这个比较靠谱）

  2. 点进去后上面应该有一个弹窗可以下载 ja-netfilter-all 或 jetbra

    若没有，去 github 搜索（原来的已经挂了-已经挂掉好多了，但应该有人备份）

  3. 在软件安装目录找到 xxx.exe.vmoptions，在最后输入：(注意斜杠方向，路径自己改)(WINDOWS 导航栏的斜杠是错的) （注：在 2021.2 版本时，IDEA 的 vmoptions 文件会在 C:\Users\ASUS\AppData\Local\JetBrains\IntelliJIdea2022.1 里面也有一份）

--add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED

-javaagent:E:\Java Dev Tools\ja-netfilter-all\ja-netfilter.jar=jetbrains

-javaagent:/path/to/ja-netfilter.jar=jetbrains

还是用自动的脚本吧，这个斜杠真的不知道应该朝哪边

参考 2：ja-netfilter-all/readme
