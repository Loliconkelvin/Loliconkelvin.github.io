Java集合框架

集合是什么

（java.util.*）集合类似于数组，数组是单个对象的容器，集合是多个对象的容器

  • 集合和数组的区别：
    1. 数组长度固定，集合的长度不固定
    2. 数组可以存储基本数据类型和引用类型（要存储基本类型可以使用装箱）

Iterator迭代器

迭代器是专门用来遍历集合的工具，可以通过Conlection及其子类对象.Interator()获取对象的迭代器

Iterator迭代器方法：

迭代器被创建时会指向第一个元素

  1. hasNext()：若当前指向有元素，返回true

  2. Next()：返回迭代器现在指向的元素，并将指向往后移

  3. remove()：从迭代器指向的conllection对象中移除迭代器返回的最后一个元素

    • 注意1：每执行一次Next之后，只能执行一次remove
    • 注意2（重要）：在迭代过程中不能使用集合对象中的移除方法，只能使用迭代器的移除方法，否则迭代器会停止迭代并抛出并发修改异常

List实现类概括

ArrayList：

  • 数组结构实现（是连续的空间），查询快，增删慢（增删会导致部分元素前后移动）
  • JDK1.2引入，效率快，线程不安全

Vector：

  • 数组结构实现，查询快增删慢
  • JDK1.0引入， 效率慢，线程安全

LinkedList：

  • 链表结构实现（非连续空间），查询慢，增删快

实现对象内容相同移除方法

在调用remove方法时实际上是先调用传入参数的类里的equals方法来遍历比较参数与集合对象的地址是否相同，并且移除第一个equals方法返回true的地方

于是我们只需要在传入的参数的类中重写equals方法即可改变remove方法的判断条件

@Override
public boolean equals(Object obj) {
    if (this == obj) {
        return true;
    }
    if (obj == null) {
        return false;
    }
    if (obj instanceof Student) {
        Student student = (Student)obj;
        // 对主键值进行比较
        if (this.name.equals(student.getName())) {
            return true;
        }
    }
    return false;
}

ArrayList源码分析

常量：

  1. DEFAULT_CAPACITY = 10：默认容量
  2. elementData：存放元素的数组

实例变量：

  1. size：实际元素个数（实例变量，实例被new出时初始化为0）（非局部变量都会被初始化）

方法：

public ArrayList() {
    // 初始化存放元素的数组（this.elementData = {}）
    this.elementData = DEFAULTCAPACITY_EMPTY_ELEMENTDATA;
}

public boolean add(E e) {
    // modCount++;
    add(e, elementData, size);
    return true;
}

private void add(E e, Object[] elementData, int s) {
    // 当前需要存储的元素将会存储在下标为s的位置上
    // 若需要存储的下标 与 数组的长度相同，扩展数组
    if (s == elementData.length)
        elementData = grow();
    // 若需要存储的下标 与 数组的长度不相同，则表明数组在该元素增加后还有空位置，直接将元素赋给对应下标的值，下标 + 1
    elementData[s] = e;
    size = s + 1;
}

private Object[] grow() {
    // 传入（再往数组里传一个对象所）需求的最小容量（下标 + 1：数组的长度 = 最大下标 + 1）
    return grow(size + 1);
}

private Object[] grow(int minCapacity) {
    // 对数组进行扩容，扩容量由newCapacity()决定
    return elementData = Arrays.copyOf(elementData,
                                  newCapacity(minCapacity));
}

private int newCapacity(int minCapacity) {
    // 旧容量 = 当前数组长度
    int oldCapacity = elementData.length;
    // 新容量 = 旧容量 * 1.5（向下取整）
    int newCapacity = oldCapacity + (oldCapacity >> 1);
    // 若 新容量 - 最小容量 <= 0, 旧容量的1.5倍居然不够用或刚好够用
    if (newCapacity - minCapacity <= 0) {
        // 若数组是空数组，返回DEFAULT_CAPACITY = 10（此时将直接将数组扩容10个单位）
        if (elementData == DEFAULTCAPACITY_EMPTY_ELEMENTDATA)
            return Math.max(DEFAULT_CAPACITY, minCapacity);
        // 不可能的值
        if (minCapacity < 0) // overflow
            throw new OutOfMemoryError();
        // 返回需求的最小容量（什么时候才会用到这个？）
        return minCapacity;
}
    // 返回newCapacity新容量（因为MAX_ARRAY_SIZE非常大，基本上判断条件为true）
    return (newCapacity - MAX_ARRAY_SIZE <= 0)
        ? newCapacity
        : hugeCapacity(minCapacity);
}

**LinkedList源码分析**

<span class="red bold">实例变量：</span>

1. ```size```：链表长度
2. ```first```：第一个元素指针
3. ```last```：最后一个元素指针

<span class="red bold">方法：</span>

```Java
private static class Node<E> {
    // 存储的元素
    E item;
    // 该数据在链表中的后一个元素
    Node<E> next;
    // 该数据在链表中的前一个元素
    Node<E> prev;

    Node(Node<E> prev, E element, Node<E> next) {
        this.item = element;
        this.next = next;
        this.prev = prev;
    }
}

public LinkedList() {
}

public boolean add(E e) {
    linkLast(e);
    return true;
}

void linkLast(E e) {
    // 定义局部变量 l = 末指针元素
    final Node<E> l = last;
    // 新建Node对象，其数据为传入的元素
    // 其前一个元素为l（last）
    // 其后一个元素为空（因此它是链表中最后一个元素）
    final Node<E> newNode = new Node<>(l, e, null);
    // 定义新元素为链表的最后一个元素（将末指针指向它）
    last = newNode;
    // 若新增的元素前面没有元素，将头指针也指向它
    if (l == null)
        first = newNode;
    // 否则将之前的最后一个元素的next指针指向新增元素
    else
        l.next = newNode;
    // 增加链表长度
    size++;
    // modCount++;
}

泛型

Java泛型是JDK1.5中引入的新特性，泛型的本质是参数化类型，把类型作为参数传递

常见的形式有泛型类、泛型接口、泛型方法

语法：

// T为类型占位符，表示一种引用类型
<T>

泛型类

使用以下格式创建一个泛型类，并且创建一个泛型变量

注意：泛型在没有给予指定类型之前不能被实例化

public class Test<T> {
    T t;

    public void showT() {
        System.out.println(t);
    }
}

使用一下格式创建一个泛型类对象

注意：泛型仅允许传入引用类型，要传入基本类型可以使用包装类

public static void main(String[] args) {
    Test<String> stringTest = new Test<>();

    stringTest.t = "123";
    stringTest.showT();
}
