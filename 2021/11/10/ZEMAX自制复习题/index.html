ZEMAX自制复习题

发表于 2021-11-10 更新于 2021-11-11 分类于 ZEMAX

13道ZEMAX题

1. 单透镜

EPD = 20mm，F/# = 10，FFOV = 10，波长，原始数据见P232~234

  • 将STO的厚度与焦距进行一个光斑的优化（无条件）
  • （改进优化）玻璃限制条件：
    • min = 2
    • max = 10
    • edge = 2
  • 将光阑移动到镜片外，进行一个优的化（无步骤提示）

最终答案见P240

2. 双胶合透镜

EPD = 50mm，F/# = 8，FFOV = 10，波长选择F,D,C，原始数据见P244

  • 光斑优化，玻璃限制条件：
    • Min = 4
    • Max = 18
    • Edge = 4
  • 空气限制条件：
    • Min = 1
    • Max = 1000
    • Edge = 1
  • 将光阑移动到镜片外，进行一个优的化

最终答案见P247

3. 牛顿望远镜

EPD = 100mm，FFOV = 4，波长选择F,D,C，原始数据见P252（容易漏数据）

  • 在STO下增加一个镜面（省略了步骤），将其设定为反射镜，反射90°
  • 在STO上面增加一个镜面，设定厚度800，将其设定为档光面（什么挡光自己选，值不提供）
  • 增加3个视场：
    • X = 0，Y=-2
    • X= 2，Y = 0
    • X= -2，Y = 0
  • 将反射镜设定为光阑（什么光阑自己选，值不提供）
  • 将第一个面设定为挡光（同上）

最终答案见P260

9. 10倍物镜

EPD = 7.76 物高Y = 0 5.6 8 波长 0.51 0.56 0.61，原始数据见P315

  • 固定焦距为16.25，修改物高Y = 0 6.5 9

  • 将OBJ厚度与第二镜面曲率设定为变量，设置默认评价函数，在DMFS上面增加以下操作数：

    操作数 Target 权重
    ISNA 0.25 1
    REAY -0.9 1
    DMFS 195 1

  • 翻转该系统，物高改为 Y = 0 0.65 0.9


  • 打开消畸变目镜，翻转该系统，设置固定焦距25

  • 打开显微镜系统，插入消畸变目镜

最终答案见P322

10. 变焦镜头（涉及多重操作数、涉及优化操作数）

  • 打开系统中的变焦镜头，设定以下数据：

    图像近轴高度Y = 0 12 17 波长选择FDC，原始数据见P264

  • 设置多重结构操作数

  • 将多种结构的数全部设为变量，曲率除OBJ、STO、IMA以外的设定为变量

  • 设定优化操作数并优化，条件：

    • 要求：三个焦距75 100 125
    • 玻璃限制：
      min = 4
      max = 10
      edge = 4
    • 空气限制：
      min = 1
      max = 1000
      edge = 1

  • 镜片大小除OBJ、STO外设为最大解


最终答案见P268

11. 扫描目镜（涉及多重操作数）

EPD = 10，无视场设置，波长 0.6328，第三镜面F/# = 10，原始数据见P271

  • 进行一次优化
  • 将光阑厚度设为50，在STO上方插入一个虚拟面，虚拟面厚度50
  • 在设定STO为折返镜，角度90°，设定第一个面为全局坐标参考面
  • 扫描全角度为40°，在第三个面实现反射镜旋转，角度自行设定
  • 设定多重结构，设定五个结构，数据自行设定
  • 进行一个优化

最终答案见P276

12. 衍射级次（涉及多重结构操作数）

EPD = 20，无视场设置，单波长

  • 在STO下方插入一个面，设定器玻璃类型为Diffraction Grating，原始数据见P129
  • 设定多重结构，级次设定为2 1 0 -1 -2
  • 打开3D图看全部组态的光线

最终答案见P131

13. 旋转角度优化方法

EPD = 20，单视场，单波长，原始数据见P141

  • 编辑坐标断点旋转，实现光束偏移3mm
  • （选做）实现玻璃片绕X轴逆时针旋转30°

最终答案见P143
