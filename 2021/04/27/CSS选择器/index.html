选中ul的第一个li元素（first-child）

  • 层次选择器:first-child 用于给层次选择器附加“第一个元素”的条件

选中ul的最后一个li元素（last-child）

  • 层次选择器:last-child 用于给层次选择器附加“最后一个元素”的条件

选择其父级元素的第n个子元素（nth-child）（按顺序选择）

  • 标签选择器:last-child(n) 用于选择隶属于该标签的父级元素下的第n个子元素
  • 该子元素必须为指定的标签才生效
  • 其n是从上往下数，并且不是指定的标签也会计入（会被其他元素阻碍）

选择器父级元素的第n个指定类型的子元素（nth-of-type）（按类型选择）

  • 标签选择器:nth-of-type(n) 用于选择隶属于改标签的父级元素下的第n个指定子元素
  • 其n只数指定的标签，非指定标签将忽略（不会被其他元素阻碍）

动作

  • hover 必须位于 link 和 visited 之后，这样样式才能生效

link

  • 设置指向未被访问页面的链接的样式

visited

  • 设置指向已被访问（点击过了）的页面的链接的样式

hover

  • 设置指向鼠标浮动在其上方时的样式
  • hover 可用于所有元素，不只是链接

active

  • 设置指向点击时的页面的链接的样式

属性选择器（常用）

  • 用于根据标签里面的属性设定选择元素

选中含有某个属性的标签

  • 若写入多个中括号，则表示选择同时含有这些属性的标签
  • 若想选中所有含有某属性的标签，可以在中括号前加星号

选中某标签为某值的元素

等于号的多种玩法（~=, |=, ^=, $=, *=）

~=（包含）

  • 属性的值中包含指定独立的单词
    [title~=flower] --> <img src="/i/eg_tulip.jpg" title="tulip flower" />

*=（包含）

  • 属性的值能组成这个单词就行
    [title*=flower] --> <img src="/i/eg_tulip.jpg" title="ffffflowerrrrrr" />

|=（开头）

  • 属性的值以指定独立单词开头，或与其他单词以空格或-分隔开
    [lang|=en] --> <p lang="en"></p> <p lang="en-us"></p>

^=（开头）

  • 属性的值的前几个字母为指定字符串就行
    [lang^=en] --> <p lang="ennn"></p>

$=（结尾）

  • 属性的值的否几个字母是指定字符串就行
    a[src$=".pdf"] --> <a src="abc.pdf"></a>
