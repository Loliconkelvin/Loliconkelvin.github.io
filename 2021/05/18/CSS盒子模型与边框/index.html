发表于 2021-05-18 更新于 2021-05-25 分类于 CSS

CSS盒子模型与边框

盒子模型

初始化操作

  • 通常在应用样式之前，我们把内外边距和下划线全部弄没：
    *{
    margin: 0;
    padding: 0;
    text-decoration: none;
}

外边距（margin）

  • 若不对body样式进行设置，body默认会有一个外边距

  • 对多个边距合并设置：

    margin: top right bottom left;

    margin: top&bottom left&right；
    只使用两个值的时候会将第一个值同时应用于上下，第二个值同时应用于左右

    margin: top left&right bottom
    只使用三个值的时候会将第二个值同时应用于左右

    margin: top&left&right&bottom
    只使用一个值的时候会将这个值同时应用于上下左右

使用外边距垂直居中

  • 行内元素不可以应用 margin 和 padding
  • 想要使行内元素居中可以：
    1. 在外面套一个div，将其宽度和高度设置为行内元素的宽高，然后使用margin
      这个是div带着里面的元素居中
    2. 在外面套一个div，然后使用 text-align: center;
      这个是div不动，里面的元素全部居中
    3. 对要居中的元素使用 display: block;,再使用 margin：0 auto;
      这个是把行元素变为块元素，然后自行居中
      div{
    margin:0 auto;
}

边框（border）

边框合并写法

border(-left/-right/-top/-bottom): border-width border-style border-color

border: 1px solid red

边框样式（style）

  • 通常设置为solid：实线
  • dashed：虚线

圆角边框（radius）

  • 应用顺序为左上 右上 右下 左下（顺时针）
  • 只应用一个值时，该值会应用于四个角
  • 应用两个值时，第一个值应用于左上和右下角，第二个值应用于右上和左下角

圆

  • 要画一个圆，需要将边框值等于正方形边长的一半

盒子的计算方式（这个元素总共多大？）

  • 这个元素的总大小为 外边距 + 边框宽度 + 内边距 + 内容宽度

边框阴影/盒子阴影（box-shadow）

盒子阴影合并写法

box- shadow: x y blur（模糊距离） color
