列表与表格

发表于 2021-04-21 更新于 2021-05-17 分类于 HTML

列表与表格（本章表格章节开始不再写快速输入有关内容）

列表

有序列表（orderlist）

  • 可使用 ol + tab/enter li + tab/enter 快速输入有序列表
  • 有序列表在生成时自带序号

无序列表（unorderlist）

  • 可使用 ul + tab/enter li + tab/enter 快速输入无序列表
  • 无序列表在生成时自带原点

自定义列表（definelist）

  • 可使用 dl + tab/enter dt + tab/enter dd + tab/enter 快速输入无序列表
  • 自定义列表包含三部分：
    1. dl：总的列表标签
    2. lt：title，表示列表标题（可以有多个列表标题，对应列表下面的内容）
    3. dd：列表内容

表格

  • 表格包含三个内容：
    1. 总表格标签（table）
    2. 行标签（tablerow）
    3. 列标签（td）
    • 列标签一定要在行标签内

▽跨行与跨列都要在列标签中操作

跨行

  • 使用 rowspan="要跨的行数" 来实现跨行
  • 在跨行之后在剩余每行删除已跨的行数

跨列

  • 使用 colspan="要跨的列数" 来实现跨列
  • 在跨行之后在该行删除已跨的列数

border值

  • 把 border="1px" 应用到table标签，可以实现边框
